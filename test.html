A rural school is implementing a coding program to promote quality education (SDG4). They have 15 students participating. The school has a small solar-powered computer lab (SDG7) with 5 computers. Each computer uses 3 units of solar-generated electricity per hour for coding lessons.

Write a Python program to calculate how many hours the computer lab can run continuously on the available solar-generated electricity before running out. Please provide the numeric answer.

The answer to this question should be a number between 1 and 30.
